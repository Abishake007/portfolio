## Overview

ECOM-WEB is a responsive e-commerce web application that enables users to browse products, manage their shopping cart, and simulate an online checkout workflow.

The project focuses on clean UI structure, state management using local storage, and building reusable React components.

---

### Key Features

- Product listing page
- Add to cart functionality
- Remove and update cart items
- Checkout simulation
- Dynamic price calculation
- Persistent cart using Local Storage
- Fully responsive UI
