## Overview

Neobank is a digital banking portal that simulates real-world banking operations including account dashboard, transaction tracking, money transfers, admin management, and secure authentication workflows.

The project focuses on clean architecture, responsive design, and structured frontend development using Angular.

---

## Key Features

- Secure login authentication
- Transaction dashboard
- Monthly financial reports
- Send money functionality
- Admin management panel
- Fully responsive UI
- Component-based Angular architecture

---

## Tech Stack

# Backend
- Java
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- MySQL
- RESTful APIs
# Frontend
- Angular (Standalone Components)
- TypeScript
- Bootstrap 5
- Bootstrap Icons
